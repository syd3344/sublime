
##本地操作指令
	一、git仓库命令
	1、git ststus — — 查看状态
	2、git add — — 工作区提交到到暂存区并跟踪
	3、git commit -m "" — — 暂存区提交到本地仓库，并存下记录
	4、git log — — 查看日志
	5、clear — — 清屏
	6、mkdir — — 创建文件夹
	7、ls — — 查看目录
	8、gitk — — 日志图形化界面
	9、rm -rk 文件名 — — 删除不为空的文件夹
	10、git config --global user.email "" — — 用户邮箱
	11、git config --global user.name "" — — 用户名
	12、git init — — 初始化本地仓库
